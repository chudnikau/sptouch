## sptouch 3.8

# Введение 

Программный комплекс sptouch предназначена для считывания и отображения оперативных данных с приборов СПГ91, СПТ92-01, СПТ92-02, СПТ92-03, СПТ920, СПГ701, СПЕ540, СПГ702, СПТ940, СПТ960, СПЕ541, СПГ705, СПГ706, СПТ961, СПГ761, СПГ763, СПЕ542, СПГ762, СПТ961М и др.

# Возможности sptouch

sptouch состоит из серверной и клиентской части. Серверная часть опрашивает и сохраняет считанные с приборов значения, а клиентаская часть отображает их.
